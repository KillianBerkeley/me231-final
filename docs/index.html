💡 Key Findings

Insights from training and evaluating all five approaches on 150,000 employee records.

  • 🏆
    Linear SVM achieved the highest accuracy (93.4%) Despite being a simpler model than the neural network or boosted tree, the linear SVM topped all classifiers — suggesting the burnout classes are nearly linearly separable in the scaled feature space.
  • 🌳
    Boosted Tree placed a close second (92.84%) HistGradientBoostingClassifier with early stopping matched SVM performance closely, and required no manual feature scaling thanks to its tree-based splits.
  • 🔵
    K-Means failed as a burnout classifier (ARI = 0.076) Extremely low Adjusted Rand Index and Silhouette Score indicate that burnout level does not cluster naturally in this feature space — the boundaries between Low/Moderate/High are not driven by geometric separation.
  • 📈
    Linear Regression explains 73.9% of burnout score variance (R² = 0.74) The continuous burnout score has meaningful linear relationships with the input features, though unexplained variance suggests non-linear factors also play a role.
  • 🤝
    Ensemble voting did not beat the best individual model Both ensemble methods achieved 93.29% accuracy — slightly below the SVM's 93.40%. Combining a weaker model (regression predictions via binning) with stronger classifiers diluted the ensemble's performance.
  • ⚖️
    Class imbalance affects macro F1 Weighted F1 (93%+) and macro F1 (59%) diverge strongly, indicating the "Moderate" burnout class dominates predictions. Minority classes (Low, High) are harder to distinguish, likely because moderate burnout shares features with both extremes.
